A production-ready, highly optimized **Multi-Tape Turing Machine Simulator** built with a premium, cyberpunk-themed academic visualization. Designed to make complex computational theory accessible, interactive, and visually stunning.

## 🚀 Features

* **Multi-Tape Architecture**: Supports complex, high-performance Turing Machine simulations with multiple independent tapes.
* **Cyberpunk UI/UX**: A sleek, dark-mode, neon-accented graphical interface that brings theoretical computer science to life.
* **Interactive Visualization**: Watch read/write heads dynamically transition across tapes in real-time.
* **React Flow State Diagrams**: Automatically generated, smoothly animated state transition diagrams. Exportable to high-res PNG!
* **Built-in Examples**: 10 pre-configured computational problems (e.g., binary addition, palindrome checking) to instantly demonstrate the engine's power.
* **Performance Optimized**: Built utilizing modern React (v19) concepts and Framer Motion for buttery-smooth metrics and transitions.

## 🛠️ Technology Stack

* **Frontend Framework**: React 19 + TypeScript
* **Build Tool**: Vite
* **Styling**: Tailwind CSS (v4) with custom cyberpunk theming
* **Animations**: Framer Motion
* **Node Visualization**: `@xyflow/react` for interactive state diagrams
* **Exporting**: `html-to-image` for generating high-quality PNGs of states

## 💻 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tavishraj/Multi-Tape-TM-Visualizer.git
   cd Multi-Tape-TM-Visualizer
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🧠 How It Works

The engine dynamically parses a Turing Machine's configuration (states, alphabet, blank symbols, transition function) and translates the computational steps into visual nodes and tape shifts. Using the **Metrics Panel**, you can track execution speed, steps taken, and computational efficiency in real-time.
